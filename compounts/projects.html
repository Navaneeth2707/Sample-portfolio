Project 2:

To-do-list

Developed a responsive To-Do List Application using HTML, CSS, EJS, and PostgreSQL, designed to enhance task management by categorizing tasks into daily, weekly, and monthly lists. The application features a clean, intuitive interface and dynamic content rendering with EJS, allowing users to easily add, edit, delete, and view tasks. PostgreSQL ensures efficient data storage and management, while the combination of Node.js and Express handles the server-side logic, providing a seamless user experience.

Front-end: HTML, EJS, CSS, JavaScript.

Back-end: Node.js, Express.

DataBase: pgAdmin4.
